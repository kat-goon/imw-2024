Project 1: Storybook Process

Introduction

Going into this project, I chose to go off of the prompt about daydreaming. Inspired by my mouse pad and the doodle I made for the IMW page, I decided to make the character a Shiba Inu. I'm new to coding, so I wanted to keep the imagery simple, so that I could focus on learning how to make forms. The Shiba Inu is on a mission to become "stylish," so you must help it navigate to the boutique and choose an accessory for it.

Storyboard

The first iteration of this project, shows my day dream of living in a penthouse in a big city. However, the storyboard I created was not as interactive as I wanted and would not induce more intuitive actions from the user.

Eventually, I decided to simplify things because drawing a city seemed too complicated for my current skill level. I then changed my idea from living in a penthouse to shopping, to simulate the dress up games I played as I child (shoutout to barbie.com!). This allowed me to make a more simple approach while focusing on learning the syntax to improve my CSS and HTML knowledge.

Inspiration and Code

Despite my sentiment of not making this project too complicated, I did not realize how complicated it would be to draw a dog in CSS, so I referenced code for a corgi drawing I found on Code Pen. Using this code and altering it to turn the corgi into a shiba inu helped me learn layering and positioning in CSS and HTML. As I progressed throughout the project, I became more comfortable with writing my own code.

Another difficult element to draw was the boutique. I searched for vector and flat illustrations so that it would be easier to visualize the shapes within. I ended up using an illustration of a coffee shop but deviated a bit from it's original aesthetic to fit my own.

While coding, there was quite a few moments where I had to trouble shoot my code. Often times it was just forgetting what I named an element or not listing it in my HTML file. To check out problematic areas, I used inspect element on my browser.

The Final & Reflection

The final product of this project is a game-like story where your choice will determine your ending. It is a simple storyline but I learned a lot from the process.

My biggest takeaways from this is that my current process of writing code is not so proficient and that when there is a will there is a way. While looking at other people's codes and while coding, I realized that there are easier ways to expedite writing the code so that it is not too redundant. Going forward, I want to familiarize myself with these shortcuts so that I become more efficient.

Overall, I am satisfied with what I created, I learned a lot about the possibilities of drawing in CSS and knowing that there is even more beyond what I currently know is an exciting thought. While I sometimes did get frustrated troubleshooting, I found the process of creating this CSS HTML storybook engaging.
